# 📁 AWS S3 File Uploader

This Python script uploads a file to an AWS S3 bucket using the `boto3` library.

---

## ✅ Requirements

- Python 3.6 or above
- AWS credentials (Access Key & Secret Key)
- boto3 library

---

## 📦 Installation

Install the required library using pip:

```bash
pip install boto3
